# Page transition made with Next.JS, Tailwind and Framer motion

This is a simple project of a page with animated transitions to practice Next.JS, Tailwind and Framer motion.

## Screenshot

[next-page-transition.webm](https://user-images.githubusercontent.com/101352684/212792811-241e3934-0549-43e5-bdc9-84c8eb6d50a6.webm)

## Built with

- [Next.JS](https://nextjs.org/) - React framework
- [Tailwind](https://tailwindcss.com/) - CSS framework
- [Framer motion](https://www.framer.com/motion/) - React animation library
